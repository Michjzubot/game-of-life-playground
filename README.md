# Game of Life Playground

Mobile-friendly Conway's Game of Life, built with vanilla HTML/CSS/JS.

## Features
- Tap/click to toggle cells on the board (canvas-based, performant)
- Play/Pause, Step, Clear, Random
- Adjustable grid size (10–120) and speed (50–800 ms)
- Responsive UI with dark styling (mobile-optimized)

## Run locally
Open `index.html` in your browser.

## Deploy (GitHub Pages)
Push to `main`, and enable GitHub Pages with source `main` / path `/`.
